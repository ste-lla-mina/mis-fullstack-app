# 🎓 Modern MIS Portal
> A sleek, role-based Management Information System for modern educational institutions.

Built with the **MERN Stack** (MongoDB, Express, React, Node.js), this platform provides a tailored experience for Admins, Teachers, and Students with a focus on clean UI and secure data management.

---

## ✨ Key Features

### 🛠 Admin Dashboard
* **Centralized Control**: Manage the entire student directory.
* **Student Registration**: Create new student records with ease.
* **System Integrity**: Full authority to delete or modify records.

### 🍎 Teacher Portal
* **Class Management**: View assigned classes and subjects.
* **Student Insights**: Real-time list of students per class.
* **Clean Tracking**: Minimalist interface for classroom focus.

### 📖 Student View
* **Academic Snapshot**: View personal profile and academic status.
* **Modern UI**: Fully responsive design for mobile and desktop.
